# Document Phase 0 CLI usage and limitations


## Description

撰寫 Phase 0 CLI 使用說明，包含 `rootDesc.xml` URL 如何取得、如何 browse、如何回報相容性問題。

## Acceptance Criteria

- `docs/phase0-cli.md` 存在。
- 文件包含 MiniDLNA / Jellyfin / Plex 測試建議。
- 文件說明目前不支援 SSDP discovery。
- 文件說明跨 VLAN / VPN 不影響手動 URL，但 discovery 之後可能受影響。
