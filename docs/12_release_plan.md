# Release Plan

## v0.0.1

目標：Phase 0 CLI PoC。

內容：

- CLI browse command。
- JSON output。
- 基本 fixtures tests。
- docs/phase0-cli.md。

## v0.1.0

目標：第一個 foobar2000 macOS MVP。

內容：

- macOS component 可載入。
- Preferences 可新增 server。
- Browser panel 可 browse。
- 可加入 playlist。
- 可播放 HTTP resource URL。
- 文件說明限制。

## v0.2.0

目標：metadata 與播放體驗改善。

內容：

- metadata mapping。
- duration。
- protocolInfo parser。
- multiple resource selection。
- basic album art URL parse。

## v0.3.0

目標：相容性與可靠性。

內容：

- MiniDLNA / Jellyfin / Plex compatibility。
- pagination。
- async loading。
- debug logging。
- troubleshooting report。

## v0.4.0

目標：SSDP discovery。

內容：

- M-SEARCH。
- discovery UI。
- manual fallback。
- 跨網段限制文件。
